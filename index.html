Weather App

I designed and built the weather app during my Shecodes Plus Workshop. It was a fascinating workshop where I learned how to integrate weather API which included daily weather forecasts and current location weather in real time. Vanilla JavaScript ES6 is like a hidden blue print that has its own mapping language called the debugger. Learning how to debug enhanced my way of finding patterns, tracking functions and making connections to end up with a well wielded application. It was exciting to also commit and push my changes on github as well as host my project on netlify to showcase my work to the community, family and friends.

Origami Website

My origami website was the very first website I ever built which was during my Shecodes Basics Workshop. My love for paper folding origami figures and crafting is what inspired me to build it. At that time I was learning how to align texts and images as well as styling them with HTML5 and CSS3. I used JavaScript ES6 to make alerts, use variables, and call functions to make it all more interactive with the user.
